Every design decision must answer: "Does this make Z3Connect look like the most premium software company in Tamil Nadu?" If the answer isn't a clear yes, cut it.

## Brand Identity
- Black-White-Black alternating section rhythm (like breathing)
- Typography: Clash Display for headings, Satoshi for body, JetBrains Mono for code/numbers
- Colors: Pure black #000 + Pure white #FFF + one blue accent #2563EB (used sparingly, only for links and CTAs when needed)
- Glass morphism: backdrop-filter blur(20px), rgba backgrounds, thin white borders at 0.08 opacity
- Animations: 0.6-0.8s for reveals, 0.3s for hovers, cubic-bezier(0.16, 1, 0.3, 1) easing, NEVER linear
- Mobile-first always (Tamil Nadu MSME audience is 90%+ mobile)
- Touch targets: minimum 48x48px
- Premium = generous spacing, breathing room
- Headlines: no jargon (no "MSME", "ERP", "SaaS" in headlines), short, punchy, confident, written for shop owners not tech audiences
- Inspiration sites: linear.app, vercel.com, raycast.com, resend.com, stripe.com, apple.com

## NEVER Use
- Caveat/script fonts — use body font instead
- Noise overlay (.noise div) — performance killer
- Mouse follower cursor — dated
- Outline text (.text-outline) — hard to read
- Purple/pink gradients — pure black/white with minimal blue
- Multiple colored accents per card — ONE accent color system
- Uppercase nav links — sentence case is more premium

## ALWAYS Include
- WhatsApp floating button (bottom-right, always visible) — wa.me/919360220928
- "Built in Tamil Nadu" badge — local pride, trust signal
- Trust badges: "30+ Businesses Trust Us", "Active Since 2022"
- Real product screenshots in device mockups
- Custom ::selection { background: #2563EB; color: white }
